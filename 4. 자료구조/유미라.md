# 1. 자료구조의 큰 그림
## 자료구조와 알고리즘
- **자료구조** : 데이터를 효율적으로 저장하고 관리하기 위한 방법
- **알고리즘** : 어떠한 목적을 이루기 위한 효율적인 연산 방법

## 시간 복잡도와 공간 복잡도
### 시간 복잡도
- 입력의 크기에 따른 프로그램 실행 시간의 관계 -> 연산 횟수

- 프로그램이 실행되었을 때 필요한 메모리 자원의 양
